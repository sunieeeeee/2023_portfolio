루미오

skill

  • H

    HTML

    웹 표준과 웹 접근성을 준수하는 HTML 정보 구조설계로 웹 사이트 퍼블리싱을 해오고 있습니다. 특히 웹 표준을 위해 HTML5에서 권장하는 마크업 방식과 시멘틱 태그를 사용해 전체 레이아웃을 설계합니다. 웹 UI 디자인 작업 후 클래스 네임과 함께 HTML 정보 구조설계를 꼼꼼히 작업한 후 퍼블리싱 작업을 진행합니다.

    C

    CSS

    크로스브라우징에 대해IE 지원을 고려하며 CSS3의 구조와 속성을 이해하고, 원하는 스타일로 코드 구현이 가능합니다. CSS3에서 새롭게 지원하는 속성을 활용해서 다양한 CSS배치와 디자인을 합니다.

    JS

    JAVASCRIPT

    자바스크립트의 기초부터 모던 자바스크립트까지 모두 익혔습니다. 네비게이션, 탭 메뉴, 어코디언등의 필수적인 코드작성이 가능합니다.

  • SC

    SCSS

    scss 사용목적을 충분히 이해하여 css 내 반복되는 내용을 줄이고 보다 효율적으로 스타일 시트를 관리할 수 있습니다. 유지보수가 편하도록 import, use를 사용하여 스타일의 쓰임에 따라 파일을 분리하여 모듈화 할 수 있습니다.

    B

    BOOTSTRAP

    부트스트랩의 문법, 규칙을 익혔으며 간단한 모듈부터 웹사이트까지 제작합니다. 클래스 네임을 사용하여 반응형 제작이 가능합니다.

    JQ

    JQUERY

    네비게이션, 탭 메뉴, 어코디언 등 웹 퍼블리싱에 반드시 필요한 상호작용을 위한 제이쿼리는 플러그인을 사용하지 않고 순수한 제이쿼리 코딩으로 제작합니다.

  • OR

    ORACLE

    오라클의 문법을 익혀서 테이블 생성, 업데이트, 변경 복사가 가능합니다. 조인과 서브쿼리를 이용하여 2개이상의 테이블의 조회를 할 수 있습니다.

  • V

    B

    VISUAL STUDIO CODE, BRACKETS

    웹 퍼블리셔로서 텍스트에디터는 Visual Studio Code를 메인으로 사용하며 경우에 따라 Adobe Brackets를 병행해서 사용합니다.

    OD

    ORACEL DEVELOPER

    orcle developer를 사용하여 좀 더 쉽게 sql 문법작성이 가능합니다.

호반 리조트

OverView.

    호반기업에서 운영하고 있는 리조트의 예약사이트를 제작하였습니다. 부트스트랩을 사용하여 제작한 모달창과 daterangepicker 라이브러리를 사용하여 제작한 달력 외에 모두 하드코딩으로 구현하였습니다. 모든 페이지가 반응형으로서 모바일, 태블릿에서도 문제없이 볼 수 있도록 제작하였습니다.

WorkInfo.

  • 제작기간2021.12~2022.01(2개월)
  • 제작영역사이트의 전반적인 퍼블리싱 담당
  • 사용 툴HTML, CSS, jQuery, daterangepicker, bootstrap
